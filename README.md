# Sales Management Portal-MERN



🏗️ System Architecture
The application is built on the MERN Stack (MongoDB, Express.js, React, Node.js) to ensure scalability and high performance.

Frontend: Built with React.js and Tailwind CSS for a responsive, modern UI.

Backend: RESTful API developed with Node.js and Express.js.

Database: NoSQL MongoDB for flexible contract schema management.

Analytics: Integrated PowerBI dashboard for KPI monitoring and financial insights.

🚀 Key Features
Contract Lifecycle Management: Full CRUD operations for managing contracts.

KPI Dashboard: Real-time tracking of contract statuses (Active, Pending, Draft).

Financial Analytics: Visual breakdown of contract values by department.

Relational Mapping: Comprehensive Entity Relationship Diagram (ERD) for data integrity.

🛠️ Installation & Setup
Prerequisites
Node.js (v14+)

MongoDB Atlas account or Local MongoDB

PowerBI Desktop (for viewing analytics)

1. Backend Configuration
Navigate to the server directory: cd server

Install dependencies: npm install

Create a .env file and add your connection string: MONGODB_URI=your_mongodb_connection_string

Start the server: npm start

2. Frontend Configuration
Navigate to the client directory: cd client

Install dependencies: npm install

Launch the application: npm start
